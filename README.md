# scratchy
repl with node_modules preloaded


## installation

    $ npm install -g scratchy

## usage

```
$ scratchy
```


## programmatic usage

```js
var scratchy = require('scratchy')
scratchy(startingDirectory)
```

`scratchy` returns a node repl object. you can do with this as you please.
